# 808 Workflow - Trap Production Panel

Dedicated panel for trap and hip-hop 808 bass programming with intuitive pitch slide controls and piano roll integration.

## Features

### 1. **Octave Shift Controls**
- Quick octave up/down buttons (-2 to +2 range)
- Visual display of current octave shift
- **Keyboard Shortcuts**: 
  - `Shift + ↑` - Octave up
  - `Shift + ↓` - Octave down

### 2. **Pitch Slide Controls**
- **Slide Time**: 10-500ms (controls how fast the pitch slides)
- **Slide Amount**: 1-24 semitones (controls pitch range)
- Real-time slider adjustments

### 3. **Quick Presets**
- **Classic 808**: 50ms slide, 12 semitones (standard trap)
- **Long Slide**: 100ms slide, 24 semitones (smooth rolls)
- **Quick Hit**: 20ms slide, 7 semitones (stab hits)
- **Slow Roll**: 200ms slide, 12 semitones (R&B/melodic)

### 4. **Piano Roll Overlay**
- Full piano roll editor for drawing MIDI notes
- Visual pitch slide curves
- 16th note grid with 4-beat emphasis
- Keyboard visualization (left side)
- **Access**: Double-click region or click "Piano Roll" button

## Usage

### Opening the Panel
1. Switch timeline to **Pattern Mode** (Audio/Pattern toggle)
2. Navigate to the **808 tab** in the right browser panel
3. Select a region to enable controls

### Creating 808 Patterns
1. Create a new audio/MIDI track
2. Draw or record 808 hits on the timeline
3. Select regions and use **Ctrl+B** to create pattern
4. Adjust pitch slides and octaves in the 808 panel
5. Double-click region to open piano roll for detailed editing

### Drawing Pitch Slides
1. Open piano roll overlay
2. Click to place notes
3. Drag between notes to create pitch slides
4. Use snap grid for rhythmic placement

## Color System
- **808s category**: Deep purple (`hsl(280 90% 65%)`)
- Used for all 808-specific patterns and controls
- Matches trap production aesthetic

## Integration with Pattern System
- 808 patterns can be saved and reused
- Drag patterns from pattern browser to timeline
- Make patterns unique to edit independently
- All 808 settings are stored per-pattern

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Shift + ↑` | Octave up |
| `Shift + ↓` | Octave down |
| `Ctrl + B` | Create pattern from selection |
| `Double-click region` | Open piano roll |
| `←/→` | Navigate timeline |

## Tips for Trap Production
1. **Classic 808 Slides**: Use 50ms slide time for authentic trap sound
2. **Sub Bass Focus**: Keep octave shift at -1 or -2 for deep bass
3. **Layering**: Create multiple 808 patterns at different octaves
4. **Pattern Variants**: Use unique patterns for verse/chorus variations
5. **4/8-Bar Loops**: Align patterns with color-coded 4-bar (cyan) and 8-bar (pink) markers
