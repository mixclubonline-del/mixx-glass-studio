# Fonts Directory

This directory contains self-hosted fonts to remove external dependencies.

## Orbitron Font

The Orbitron font files should be placed here:

- `orbitron-v31-latin-regular.woff2` - Regular weight (400)
- `orbitron-v31-latin-regular.woff` - Regular weight fallback
- `orbitron-v31-latin-700.woff2` - Bold weight (700)
- `orbitron-v31-latin-700.woff` - Bold weight fallback

## Download Instructions

1. Visit [Google Fonts - Orbitron](https://fonts.google.com/specimen/Orbitron)
2. Download the font family
3. Extract the WOFF2 and WOFF files for weights 400 and 700
4. Place them in this directory with the names above

Alternatively, use a tool like `google-webfonts-helper` or download directly from the Orbitron GitHub repository.

## Benefits

- Removes external dependency on Google Fonts
- Faster loading (no external request)
- Better privacy (no tracking)
- Works offline
- Complete control over font delivery
